# ERD - Приложение для работы с данными

## Описание
ERD - это многофункциональное приложение, разработанное на Flutter для работы с данными и их визуализации. Приложение поддерживает работу на различных платформах, включая Windows и Android.

## Технические характеристики
- Версия: 0.1.0
- SDK: Dart >=3.1.3 <4.0.0
- Фреймворк: Flutter

## Основные возможности
- Работа с базами данных SQLite
- Визуализация данных с помощью Syncfusion Charts
- Поддержка Excel файлов
- Интеграция с Firebase
- Работа с таблицами данных (DataGrid)
- Кроссплатформенная поддержка

## Зависимости
Основные библиотеки:
- sqflite: ^2.3.0 - для работы с SQLite
- syncfusion_flutter_charts: ^29.1.35 - для визуализации данных
- firebase_core: ^3.13.0 - интеграция с Firebase
- excel: ^4.0.2 - работа с Excel файлами
- file_picker: ^10.1.2 - выбор файлов
- path_provider: ^2.1.1 - работа с файловой системой

## Установка и запуск
1. Убедитесь, что у вас установлен Flutter SDK
2. Клонируйте репозиторий
3. Выполните команду `flutter pub get` для установки зависимостей
4. Запустите приложение командой `flutter run`

## Разработка
Для разработки рекомендуется использовать:
- Visual Studio Code или Android Studio
- Flutter Dev Tools
- Firebase Console (для работы с Firebase)
